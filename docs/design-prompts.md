# Claude 设计提示词 · 全部页面/功能

用法:**先粘「通用设计简报」,再粘某个页面的提示词**,发给 Claude(建议触发 frontend-design 技能,或在 Claude.ai 用 artifact 出 HTML 预览)。产出可以是「独立 HTML mockup」用来看效果,或「Astro 组件/页面」直接进项目(在提示词末尾注明你要哪种)。

---

## PART 0 · 通用设计简报(每次先粘这段)

```
你在为我的个人网站「记录我的一生」做 UI 设计。整体气质:动物森友会式 cozy、温暖、亲和、可爱但不幼稚,信息层次清晰。

技术约束:
- Astro 静态站,纯展示;交互用少量原生 JS(群岛)。
- 颜色/字体/圆角/阴影一律用 CSS 变量(主题 token),不要写死颜色,这样能换肤。
- 移动端优先、响应式;无障碍(语义标签、对比度、focus 可见)。

主题 token(整站靠 <html data-theme> 切换,组件只用变量):
- cozy(默认):--bg:#fdf3df; --surface:#fff8e7; --ink:#794f27; --muted:#9f927d; --accent:#19c8b9; --radius:18px; --shadow:0 4px 12px rgba(61,52,40,.1); --font:"Nunito","Noto Sans SC"。
- editorial:--bg:#faf8f4; --surface:#fff; --ink:#1a1a1a; --accent:#1a1a1a; --radius:2px; --shadow:none; --font:Georgia,"Noto Serif SC"。大留白、衬线、细线条。
- scrapbook(规划):--bg:#efe6d3; --ink:#7a5c3a; --accent:#e08a5a; --radius:6px;手写体点缀、胶带、拍立得。
- dark(规划):--bg:#0e1116; --ink:#e7e9ee; --accent:#5eead4(辅 #a78bfa);等宽点缀、霓虹发光、节点连线。

复用组件类:.container(居中容器,max-width 用 --maxw)、.card、.btn(胶囊主按钮)、.muted、.prose(正文排版)。

设计时:用 cozy 主题呈现默认稿,但确认在 editorial 下也成立(即只靠变量、不靠固定色)。给我:布局结构、各区块、关键组件、交互与状态(hover/active/空状态/加载)、响应式断点要点。先给低保真结构,再给一版精修。
```

## PART 1 · 第一阶段页面(已上线)

### 全局 · 顶部导航 + 主题切换 + 页脚
```
设计整站通用的「顶部导航栏 + 主题切换器 + 页脚」。
- 顶栏:左侧 Logo(🏝 + 站名);中间主导航(关于/人生时间轴/博客/项目);「更多 ▾」下拉容纳其余模块;右侧主题切换器。常驻、半透明毛玻璃可选。
- 主题切换器:每个主题一个彩色小圆点(cozy 薄荷/editorial 黑/scrapbook 橙/dark 青),当前项有 outline;点选即换肤并记忆。
- 页脚:版权 + 一句话签名。
- 移动端:主导航收起为汉堡或仅留切换器;给出窄屏方案。
要 Astro 组件(TopNav.astro / ThemeSwitcher.astro / Footer)。
```

### 首页 · 氛围入口枢纽
```
设计个人网站首页,定位是「一生的索引 / 发射台」。
- 顶部 hero:圆形头像 + 「你好,我是 ___」大标题 + 一句话介绍;氛围轻盈。
- 模块卡片网格:每张卡 = 一个模块(图标 emoji + 名称),点击进入;未启用模块灰显「敬请期待」且不可点。
- 可选:一句当前状态 / 最近更新条;以及一个进入「3D 世界模式」的入口按钮。
- 网格响应式:桌面 3 列、平板 2 列、手机 1–2 列。
卡片 hover 上浮微动。要 cozy 默认稿 + 它在 editorial 下的样子。
```

### 关于我
```
设计「关于我」页。
- 头部:头像 + 名字 + 头衔。
- 正文:一段自我介绍(Markdown 渲染,用 .prose)。
- 社交链接:一排胶囊按钮(GitHub/邮箱/X/博客…,带 emoji 图标)。
- 可选小区块:此刻在做什么、所在城市、兴趣标签。
单列、留白舒适、像一张温暖的名片。
```

### 人生时间轴(主轴)
```
设计「人生时间轴」页,这是整站记录一生的脊梁。
- 竖向时间线:每个事件含 日期、类型彩色圆点 + emoji 图标、标题、摘要;project 类事件可「→ 查看」跳转。
- 顶部类型筛选 chips:全部/人生/项目/足迹/音乐/里程碑;点选只显示该类型(纯前端切换),当前 chip 高亮。
- 事件按时间倒序;支持很多事件时的可读性(分年份分隔可选)。
- 空状态、移动端单列。
数据形如 { date, title, type, icon, link?, 摘要 }。要时间线在 cozy 与 editorial 下都好看。
```

### 博客 · 列表
```
设计「博客列表」页。
- 文章卡片列表:标题、日期、标签;过滤草稿;按日期倒序。
- 可选:置顶/精选、标签筛选、年份分组。
- hover 微动;空状态;移动端单列。
卡片用 .card,标题为主、元信息 .muted。
```

### 博客 · 文章详情
```
设计「博客文章详情」页,阅读体验优先。
- 文章头:标题、日期、标签;可选封面图、阅读时长。
- 正文:Markdown 排版(.prose):标题层级、引用、代码块、图片、列表都要好看且舒适行宽。
- 文末:返回博客、上一篇/下一篇(可选)。
editorial 主题下应呈现杂志级排版。
```

### 项目 · 列表
```
设计「项目列表」页。
- 卡片网格:每张含 标题、一句话描述、标签;可选封面缩略、角色、年份。
- 桌面 2 列、手机 1 列;hover 上浮。
- 可选:按标签/年份筛选。
卡片整体可点进详情。
```

### 项目 · 详情
```
设计「项目详情」页。
- 头部:标题、角色、时间、标签;主操作「打开项目 ↗」胶囊按钮。
- 媒体:封面/截图画廊(可选,占位即可)。
- 正文:项目说明(.prose);可加「我做了什么 / 技术栈 / 成果」小节。
- 文末:返回项目列表。
```

## PART 2 · 后续模块(规划中,按 spec 路线图)

### 音乐(后续叠加播放器)
```
设计「音乐」页:cozy 唱片墙气质。
- 区块:正在循环、年度专辑、我的歌单(每个歌单一张卡:封面、名称、曲目数)。
- 单曲/专辑卡:封面、标题、艺人、可选评分;hover 浮起。
- 预留「内嵌播放器」位:歌单卡上的播放按钮 + 底部 mini 播放条(进度、上一首/下一首、当前曲目),先做静态 UI + 状态(播放/暂停)。
- 数据形如 music.yaml:playlists[], favorites[], nowPlaying。
```

### 游戏 · 通关墙
```
设计「游戏」页:像一面成就/通关墙。
- 游戏卡网格:封面、名称、平台、状态(在玩/通关/搁置)、我的评分(★)、时长。
- 顶部按状态/平台筛选;统计条(共玩 N 款、通关 M 款)。
- 通关的卡加一枚「✓ 通关」徽章;hover 翻看一句短评。
```

### 影视 / 剧(后续叠加预告片/播放器)
```
设计「影视清单」页:海报墙。
- 海报网格卡:海报、片名、年份、我的评分、状态(想看/在看/看过)。
- 顶部筛选(电影/剧/想看/看过);hover 显示一句短评。
- 预留交互:点海报弹出详情抽屉(简介 + 「看预告片」嵌入位 + 我的评论)。
- 数据 films.yaml。
```

### 足迹地图
```
设计「足迹」页:去过的地方。
- 主体:一张地图(可用占位)上打点;点位 = 城市/地点。
- 侧栏/下方:地点列表卡(城市、日期、一句话、照片缩略);点列表项地图聚焦。
- 顶部统计:去过 N 城 / M 国;按时间或地区筛选。
- 移动端:地图在上、列表在下。
```

### Now · 我现在在做什么
```
设计「Now 页」(借鉴 nownownow.com):
- 一段「最近在忙什么」自述;分小块:在做的项目、在读、在学、在听、近期目标。
- 顶部标注「最后更新于 X」。
- 简洁单列、亲切口吻、易于频繁更新。
```

### 人生统计仪表盘
```
设计「人生统计仪表盘」:把一生数据可视化。
- 卡片式指标:听过 N 首歌、通关 M 款游戏、去过 K 座城、写了 P 篇、活了 D 天…大数字 + 小图标。
- 若干小图表(占位):听歌趋势、足迹年份分布、博客频率。
- 顶部一句「我的一生,用数字」。要有信息密度但不杂乱;cozy 与 dark 下都要炫。
```

### 日记(公开)
```
设计「公开日记」页:比博客更随性、更私人。
- 时间流:每条 = 日期 + 心情 emoji + 短文 + 可选配图;像时间线/微博流。
- 顶部按月份/心情筛选;支持很多条时的滚动可读性。
- 语气温暖、留白足。
```

### 许愿树
```
设计「许愿树」页:一棵可视化的树,枝上挂愿望。
- 主视觉:一棵 cozy 插画风的树;每个愿望 = 挂在枝头的小卡/灯笼/果实;已实现的会「点亮/结果」。
- 点愿望弹出详情(许愿日期、进度、实现日期)。
- 顶部:已实现 X / 共 Y;新增愿望入口。
- 要有生长/点亮的小动效。
```

### 人生清单 Bucket List
```
设计「人生清单」页:这辈子想做的事。
- 可勾选清单:每项 图标 + 标题 + 可选目标日期/分类;完成的打勾 + 划线 + 完成日期 + 可附照片。
- 顶部进度条(完成 N / 共 M)+ 分类筛选(旅行/技能/体验…)。
- 完成时有小庆祝动效。
```

### 时间胶囊
```
设计「时间胶囊」页:写给未来的自己,到期才解锁。
- 胶囊卡列表:封存日期、解锁日期、收件人(未来的我/某人)、状态(封存中/已解锁)。
- 封存中:卡片上锁、显示倒计时,内容隐藏;已解锁:可展开阅读。
- 新建胶囊入口(标题、解锁日期、正文)。
- 神秘、仪式感、温暖的视觉。
```

### 相册 / 回忆墙
```
设计「相册」页:照片故事。
- 瀑布流/网格相册;可按相簿(事件/年份)分组。
- 点图进灯箱(大图 + 说明 + 日期);键盘左右切换。
- 顶部相簿切换;移动端手势友好。
```

### 年度回顾
```
设计「年度回顾」页(如「我的 2025」):
- 顶部年份切换;一屏式纵向叙事:今年关键词、大事记、最爱的歌/游戏/书、去过的地方、数字总结。
- 图文混排、有杂志感与仪式感;可分享。
```

### 播客
```
设计「播客」页。
- 节目列表卡:封面、标题、时长、日期、简介;内嵌音频播放条(播放/进度/倍速,先做静态 UI)。
- 顶部:订阅入口(Apple/Spotify/RSS 图标按钮)。
- 移动端单列、播放条吸底。
```

### 视频 / Vlog
```
设计「视频」页。
- 视频卡网格:缩略图 + 时长角标 + 标题 + 日期;点开弹出播放器(嵌入位占位)。
- 顶部分类/合集筛选。
```

### 好物收藏
```
设计「好物」页(我用过爱用的东西)。
- 卡片网格:图片、名称、类别、一句话推荐理由、可选外链「去看看」。
- 按类别筛选(数码/家居/文具…);hover 显示理由。
```

### 语录收藏
```
设计「语录」页:打动我的句子。
- 引用卡片瀑布流:大号引号、句子、出处;不同卡用主题强调色点缀。
- 可按主题/来源筛选;点击可复制。
- 留白与排版要有「值得驻足」的感觉。
```

### 留言板 / 访客
```
设计「留言板」页(静态站友好,留言可走第三方/issue 占位)。
- 留言流:头像/昵称 + 时间 + 内容;cozy 气泡卡。
- 顶部留言输入区(昵称 + 内容 + 提交,先做 UI + 提交中/成功/失败状态)。
- 友好、温暖、像小岛访客簿。
```

### 数字花园 / 知识库
```
设计「数字花园」页:互相链接的笔记。
- 入口:可视化关系图(节点 = 笔记,连线 = 关联;占位即可)+ 笔记列表。
- 笔记详情:正文 + 「相关笔记」+ 双向链接;成熟度标记(🌱萌芽/🌿成长/🌳成熟)。
- dark 主题下做出「节点发光连线」的极客感。
```

### 小游戏彩蛋
```
设计「小游戏」入口与一个示例小游戏的 UI(如猜歌/点彩蛋/Canvas 小游戏)。
- 入口卡:游戏名 + 一句玩法 + 开始按钮。
- 游戏内:简洁 HUD(分数/计时)、开始/结束/重来状态;可爱反馈动效。
- 轻量、纯前端、能撒在各页面当彩蛋。
```

### 3D 世界模式(第四阶段)
```
设计「3D 世界模式」的 2D UI 叠层(HUD/overlay),3D 场景本身另做。
- 顶部:返回「内容模式」按钮 + 主题切换。
- 进入世界的加载页(进度)+ 操作提示(移动/交互按键、移动端摇杆)。
- 靠近地标时的「按 E 查看」提示 + 弹出的项目/模块卡(复用内容模式的卡片样式)。
- HUD 要不挡视野、与主题 token 一致(含 dark 的霓虹感)。
```

## 备注
- 想要直接进项目的代码,在提示词末尾加:「输出为 Astro 组件,样式用 scoped `<style>` + 上面的 CSS 变量,内部链接用 `withBase()`。」
- 想先看效果,加:「输出为单文件 HTML mockup(内联 cozy 主题变量),用真实占位内容。」
- 数据结构以 `src/content/` 与 `src/config/site.ts` 为准;新模块沿用「Markdown/YAML + 内容集合」的方式。
